I'm a **2026 AI/ML grad** stuck on one question: *how do you get an LLM to actually do the work — not just talk about it?*

For me the answer is **agentic systems** — multi-agent orchestration, Agentic RAG, and MCP tool-calling in Python that plug into real services and ship real outcomes. I shipped production AI at **TCS**, then went further as the founder and lead engineer behind **PaperX**, an AI-native EdTech platform now used by **4,200+ students**.

I'm after **new-grad / early-career roles** where I can take agentic AI from demo to production. If that's what your team is building, let's talk.

## 🧠 What I'm building

- **PaperX** — AI-native EdTech platform (~41K LOC, 4,200+ users). AutoGen agent orchestra, Agentic Graph RAG over a verified academic knowledge graph, a 133-tool MCP server, and multi-model inference. *Founder & lead engineer.*
- **Virtu** — an AI hiring portal with an agentic screening committee (LangGraph + AutoGen) that reads real code and verifies candidates instead of keyword-matching résumés.
- **Oncexpress** — surface-logistics ERP with Amazon-grade live waybill tracking, built on FastAPI + Supabase.
